# TERMINAL SNAKE

I've always liked seeing how far I could push textbased games. Snake was always one I wanted to try.

# How to run
1. git clone "https://github.com/AHL96/Terminal-Snake"
2. cd Terminal-Snake
3. dotnet run

# Don't have dotnet core?
1. git clone "https://github.com/AHL96/Terminal-Snake"
2. cd Terminal-Snake/publish
3. ./Terminal\ Snake
